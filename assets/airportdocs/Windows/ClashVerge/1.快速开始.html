    常见问题解答

    Clash Verge 是一款功能强大的图形化代理客户端，支持多种协议。在 Windows 平台使用过程中，您可能会遇到以下常见问题。本页面为这些问题提供了解决方案，帮助您顺利使用 Clash Verge。

    1. 安装与启动问题

    1.1 安装后无法启动或提示文件损坏

    可能原因：

    • 文件损坏或缺少必要的核心组件。
    • 程序未安装到正确路径。

    解决方案：

    • 删除旧配置：卸载旧版本，确保删除残留的配置文件（如 C:\Users\<用户名>\.config\clash-verge）。
    • 重新安装：从 官方 GitHub Releases 页面 下载最新版本，重新安装。
    • 检查权限：以管理员模式运行 Clash Verge。

    1.2 启动时界面显示不全或空白

    可能原因：

    • 系统兼容性问题或缺少必要的依赖（如 WebView2）。

    解决方案：

    • 更新系统：确保 Windows 操作系统已更新到最新版本（Windows 10/11）。
    • 检查依赖：安装最新版本的 Microsoft Edge WebView2 Runtime。
    • 删除配置文件：尝试删除软件主目录下的 config.yml 文件并重启（删除前请备份）。

    2. 系统代理冲突问题

    2.1 Clash Verge 与 Clash for Windows 冲突

    现象：

    • Clash Verge 启动后无法正常启用系统代理。
    • 系统代理设置被 Clash for Windows 占用，导致 Clash Verge 无法生效。

    解决方案：

    1. 关闭 Clash for Windows 的系统代理：
    2. 打开 Clash for Windows，进入设置页面。
    3. 关闭 System Proxy（系统代理）。

    4. 确保退出 Clash for Windows 时不会残留系统代理设置。

    5. 清理残留代理设置：

    6. 按下 Win + R，输入 inetcpl.cpl 打开 Internet 属性。

    7. 进入 连接 > 局域网设置，取消勾选 为 LAN 使用代理服务器。

    8. 重新启动 Clash Verge：

    9. 以管理员身份运行 Clash Verge，重新启用系统代理。

    3. 订阅与节点问题

    3.1 导入订阅时提示“error sending request for url”或“远程主机强迫关闭了一个现有连接”

    可能原因：

    • 订阅链接被阻断或网络问题导致无法访问。

    解决方案：

    • 多次导入尝试：切换Wi-Fi/数据网络，多次导入订阅链接尝试保存。
    • 获取新的订阅链接：联系代理服务提供商，获取未被墙的新订阅链接。
    • 使用订阅转换工具：通过在线工具（如 subconverter）将订阅转换为可用格式后导入。

    3.2 无法翻墙，但国内网络访问正常

    可能原因：

    • 代理节点信息变动或节点不可用。

    解决方案：

    • 更新节点配置：刷新订阅，获取最新节点信息。
    • 检测节点可用性：在服务器列表中，选择延迟较低的节点进行连接。
    • 联系服务商：如果所有节点均不可用，联系服务商咨询线路问题。

    3.3 所有网络都无法访问，电脑断网

    可能原因：

    • Clash Verge 未正常退出，导致系统代理设置异常。

    解决方案：

    1. 重启 Clash Verge：尝试重新启动软件并启用系统代理。
    2. 检查系统代理设置：
    3. 打开 Internet 属性（按下 Win + R，输入 inetcpl.cpl）。
    4. 确认已禁用 为 LAN 使用代理服务器。
    5. 禁用系统代理后重启：
    6. 在 Clash Verge 中关闭 系统代理 设置后重新启用。

    4. 使用与功能问题

    4.1 如何选择代理模式？

    解决方案：

    • 在主界面选择代理模式：
    • 规则模式：推荐，流量根据规则分流。
    • 全局模式：所有流量通过代理。
    • 直连模式：禁用代理。
    • 在节点列表中，选择延迟较低的节点并启用。

    4.2 如何导入订阅？

    解决方案：

    • 获取订阅链接：从代理服务商处获取。
    • 导入订阅：
    • 打开 Clash Verge > 配置 > 订阅链接 > 添加订阅。
    • 粘贴订阅链接并保存。

    4.3 日志调试功能

    解决方案：

    • 在主界面点击 日志，查看详细的连接信息。
    • 在设置中调整日志级别（如切换到 TRACE 模式）以获取更详细的调试信息。

    5. 其他问题

    5.1 安装后点击图标提示“无效连接”

    可能原因：

    • 程序安装在包含中文字符的目录下。

    解决方案：

    • 重新安装：确保将程序安装到不包含中文字符的路径中，例如：C:\Program Files\ClashVerge。

    5.2 如何切换软件语言？

    解决方案：

    • 打开设置 > Verge Setting > Language，选择 中文 并保存。

    通过上述常见问题解答，您可以轻松解决在 Windows 上使用 Clash Verge 时可能遇到的问题。如果您仍有疑问，请参考官方文档或联系服务支持。
