    快速开始

    Clash Verge 是一款功能强大的图形化代理客户端，支持多种协议。以下是 Windows 平台上的基础安装与代理配置步骤，帮助您快速上手。

    1. 下载与安装

    下载软件

    1. 打开 Clash Verge rev 的 GitHub 仓库。
    2. 下载适用于 Windows 的安装文件，例如：Clash.Verge.Setup.x.x.x.exe。
    或 在GitHub的下载页面上点击已分类的快捷下载，如图所示。

    安装软件

    1. 双击下载的 .exe 文件运行安装程序。
    2. 按照提示完成以下步骤：
    3. 选择安装路径：建议保持默认路径。
    4. 完成安装：点击 下一步，等待安装完成。
    5. 安装完成后，您可以通过开始菜单或桌面快捷方式启动 Clash Verge。

    2. 初始化配置

    第一次运行软件

    1. 打开 Clash Verge，系统可能提示您允许相关权限：
    2. 网络权限：确保允许以便正常连接服务器。
    3. 系统代理权限：如有提示，请勾选“始终允许”。

    3. 导入代理配置

    Clash Verge 支持两种方式导入代理配置：通过订阅链接或本地文件。

    方法一：使用订阅链接

    1. 从您的代理服务提供商控制台获取 订阅链接，点击复制按钮复制链接。
    2. 在 Clash Verge 左侧菜单中，点击 配置。
    3. 点击 订阅链接 > 添加订阅。
    4. 粘贴订阅链接到输入框，点击 保存。
    5. 软件会自动更新订阅并显示服务器列表。

    方法二：通过本地文件导入（无关紧要）

    如果URL导入失败 可以根据这里操作
    1. 复制你的订阅链接，在订阅链接后面加入 &app=clashmeta ，然后通过浏览器打开，文件会自动下载。
    2. 下载后，请重命名文件后缀为 .yaml
    3. 点击 导入文件，选择刚才的文件。
    4. 成功导入后，您将在主界面看到服务器列表。

    4. 启用代理

    开启系统代理

    1. 点击左侧菜单 设置。
    2. 找到 系统代理，开启该选项。

    选择代理模式

    • 默认情况下，Clash Verge 使用 规则模式，推荐新手使用：
    • 规则模式下，流量会根据规则智能分流。
    • 默认采用 LoadBalance（负载均衡），以平衡节点流量和速度。
    • 如果需要切换模式，您可以在 节点选择页面 选择以下模式：
    • 规则模式：根据规则智能分流流量（推荐）。
    • 全局模式：所有流量通过代理。
    • 直连模式：禁用代理。

    切换服务器

    1. 打开主界面，找到 服务器列表。
    2. 选择你需要链接的国家/地区的代理节点。
    3. 点击节点名称以启用。

    5. 测试代理是否正常工作

    1. 打开浏览器访问 Google 或其他被墙网站，确认是否可以正常访问。
    2. 在 Clash Verge 主界面，查看服务器状态和延迟：
    3. 正常连接的服务器会显示绿色状态。
    4. 延迟数值应低于 200ms。
    5. 如果无法正常工作，请检查以下内容：
    6. 确认套餐还有流量没有被暂停。
    7. 切换其他不同的节点尝试。

    教程总结

    通过本教程，您已完成以下内容： 1. 下载并安装 Clash Verge。 2. 配置代理订阅。 3. 启用系统代理并测试连接。

    如果需要更高级的功能（如自定义规则、日志调试等）或遇到问题，请参考后续的 常见问题 和 进阶配置 部分。
